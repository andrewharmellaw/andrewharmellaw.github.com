Some Definitions

This section in Gelfand (46) begins with some definitions. I found it useful to take these and re-state them in my own terms.It was useful to have to write it, and perhaps it will help you to have an alternative too.

An Identity says “this expression is equal to that expression”. If proved (by applying standard algebraic transformations) then it holds for all values of the variables. That is also the standard mechanism by which an Identity is refuted - by finding values for the variables where the expressions are not equal.

An Equation says “these two expressions are equal”. It must be solved (i.e. you must find all the values of the variables for which the left hand side is equal to the right hand side).

Note: Mathematicians don’t say an Equation is “unsolvable”, because they deem it to be solved if they prove it can have no solutions. To “solve” therefore means to find all, or prove that there are no, solutions.
